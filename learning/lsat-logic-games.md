# LSAT Logic Games 学習記録

## 目的
- **主目的:** CoderPad論理テスト対策（Devdegree再挑戦）
- **副次効果:** DET Reading/論理問題の基礎力強化

## 戦略

### Phase 1: 新規問題を最大限解く（2-4月）
- 同じ問題の繰り返しより、**多様な問題パターンに触れる**
- 論理テストは「パターン認識」が命
- 目標: 50-100問（可能な限り）

### Phase 2: 弱点パターンの反復（5-6月）
- Phase 1で間違えた/時間かかった問題のみ再挑戦
- パターン別に弱点整理

### Phase 3: 実戦形式（7-8月）
- 時間制限付きで解く
- CoderPad形式に近い環境で練習

## 問題ソース

### CoderPad選択式論理テスト（コードなし）向け対策
- 主軸: LSAT Analytical Reasoning（Logic Games）
- 形式: 制約条件から論理的配置を導く、物語調長文 + 選択肢回答
- 外部評価: LSATスコア（180点満点）
- 無料練習: LawHubの公式練習問題・模試（無料範囲）
- 有料範囲: 正式LSAT受験は有料（目安: 約$200）

## メモ・気づき

### なぜ繰り返しより新規優先？
1. **パターン認識が重要** - 同じ問題5回 < 異なる問題5つ
2. **CoderPadは初見問題** - 慣れた問題では対策にならない
3. **時間効率** - 新鮮な刺激 > 記憶の強化

### 繰り返しが有効なタイミング
- 全パターン網羅後
- 弱点タイプ特定後
- 本番直前（記憶定着）

### 他の学習との連携
- DET Reading - 論理構造理解
- LeetCode - アルゴリズム思考
- 全て「構造化された思考」を鍛える
