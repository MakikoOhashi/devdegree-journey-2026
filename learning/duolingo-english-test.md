## 学習方針

### 迷わないための実行原則（2026-02整理）

#### 前提の分離
- 短期が効かない: 英語能力は一晩で大きく変わらない
- 短時間・高密度: 出し方（処理の型）は短期で矯正できる
- この2つは矛盾しない

#### DETの1行定義
- **DET = 制限時間下で、英語を「意味のある構造」に即変換できるかの試験**

#### 何が効くか（優先順位）
- 1位: 処理の型を固定する
- 2位: 語彙は「覚える」より「即出せる」状態にする
- 3位: 多読は長期英語力には有効だが、DET直結のROIは低め

#### 20-30分の高密度セッション
- Speaking/Writing:
  1. 立場を即決
  2. 理由は2つまで
  3. 具体例は1つ
  4. 迷ったら抽象語で構造を維持
- Reading/Listening:
  - 全理解を狙わず、主張または事実を1つ確実に取る

#### 迷った時の判断軸
- これは ①語彙量を増やす ②英語量を増やす ③処理の型を固定する のどれか？
- DETで最優先は **③**

#### Write about a photo（固定テンプレ）
- 方針: **型を使う（毎回同じ骨組み）**
- 4文テンプレ:
  1. `In the picture, I can see ...`
  2. `On the left/right, there is/are ...`
  3. `In the background, ...`
  4. `Overall, it looks ...`
- 目標: 40-60語を1分で安定して出す

#### 運用方針（面接対策と一本化）
- 平常時20分:
  - X等の英語記事で主張を1つ抽出（5-10分）
  - 英語ログを3-5文で記述（主張1 + 理由1-2）（5分）
  - 面接用説明を1本（5-10分）
- DET直前2-3週間:
  - 公式Practice週2
  - Speaking/Writingの型確認に集中

#### 最終結論
- **DETは英語力そのものを短期で伸ばす試験ではなく、既存英語力を時間内で壊さず出せるかの試験**

## 課題と対策

### 現状の課題
1. **教材が少ない**
   - 対策: LSAT、GMAT ReadingをDET対策に転用
   
2. **類題が作りにくい**
   - 対策: 公式Practiceを最低2周
   - 対策: 自分のAI類題生成プロダクトを活用（Meta認知）

3. **マイルストーンが曖昧**
   - 対策: 上記の月次マイルストーン設定
   - 対策: 週次で進捗記録

### TOEFLとの違い
| 要素 | TOEFL | DET |
|------|-------|-----|
| 教材 | 豊富（毎朝R長文1本可能） | 限定的 |
| 時間 | 3時間 | 1時間 |
| 受験 | 会場 | 自宅 |
| 結果 | 7-10日 | 48時間 |
| 対策 | 定型化しやすい | 柔軟性必要 |

### DET特化戦略
- **公式Practiceを徹底的に**: 少ない教材を最大活用
- **LSAT併用**: 論理力を根本強化
- **高速フィードバック**: 48時間結果 → 弱点即修正可能
- **複数回受験前提**: 3-4週間隔で2-3回受験想定

### 実戦フォーマット（運用）
- 主軸: 公式Practice Testを週2-3回
- 目安: スコア帯120±5に入るまで継続
- 弱点別補助:
  - Reading: TOEFL Readingで高速処理
  - Listening: 短い音声の即要約
  - Speaking: 30秒即興スピーキング
  - Writing: 5分要約 + 意見1文トレーニング

### やらないこと
- 怪しいDET専用アプリへの課金
- 形式暗記のゴリ押し
- TOEFLフル模試の回しすぎ
