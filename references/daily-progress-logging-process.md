# Daily Progress Logging Process

今後の作業記録は、以下の手順で統一する。

## 手順（毎日作業終了時）
1. 対象リポジトリのREADME差分を確認する
2. 差分を箇条書きで要約する（CHANGE）
3. 各項目に理由を記入する（WHY）
4. 日付つきでプロジェクトのDaily Decision & Failure Logに追記する

## 記録フォーマット
- CHANGE: 何を変えたか
- WHY: なぜ変えたか
